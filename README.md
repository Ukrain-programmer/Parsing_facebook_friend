# Facebook Friends Parser

This Python project is designed to parse Facebook friends sand process data using Selenium for web scraping.

## Installation

> Requires Python >= 3.8

```bash
pip3 install -r requirements.txt
```

## Usage

```bash
cd src && python3 main.py
```
